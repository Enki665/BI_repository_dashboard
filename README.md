# BI Dashboard Portfolio (Power BI + Tableau)

Este repositorio reúne una selección de tableros de control desarrollados con **Power BI** y **Tableau**, orientados a **toma de decisiones** (KPIs, análisis de desempeño y exploración de causas).  
Cada proyecto incluye: contexto del caso, preguntas de negocio, definiciones de KPIs, notas del modelo de datos y evidencias (capturas y PDF). Cuando aplica, se agrega un enlace público para visualizar el tablero.

## Enfoque de trabajo
- Diseño orientado a usuario: tableros para perfiles ejecutivos y analíticos.
- Métricas con definiciones explícitas (evita ambigüedad de KPIs).
- Modelado tipo estrella (hechos + dimensiones) cuando el caso lo permite.
- Evidencia “reclutador-friendly”: capturas y PDFs siempre incluidos.

## Proyectos destacados

### 1) Executive KPI Scorecard
**Objetivo:** vista ejecutiva de desempeño vs meta, variaciones MoM/YoY y alertas rápidas.  
**Herramientas:** Power BI | Tableau  
**Carpeta:** `projects/01-executive-kpi-scorecard`

### 2) Sales Performance Dashboard
**Objetivo:** análisis comercial por canal/región/producto, tendencias y Pareto de contribución.  
**Herramientas:** Power BI | Tableau  
**Carpeta:** `projects/02-sales-performance`
